Help

HeadVR (https://github.com/autoandshare/HeadVR) is a open source VR video player for android.

It is controlled by head motion, e.g., nod or shake of the head. (Gyroscope sensor is necessary)

It supports 2D and 3D video (360 video not supported yet). For 3D video, file name should include 'SBS' or 'TAB' (https://kodi.wiki/view/3D#Video_filenames_flags).

Currently there is no browsing function and please open a video with this app from a file manager.

 

There are 5 basic head motions: Up, Down, Left, Right, Hold still.

 

Player basic functions:

Play/Pause - Down, Up (Nod)
Forward - Right, Hold still, Left
Backward - Left, Hold still, Right

 

Playlist functions:

If the video is a local file, a play list will be created automatically with video files under the same directory.

Next Video - Up, Down, Right, Left
Prev Video - Up, Down, Left, Right

 

Change settings:

Some settings can be adjusted inside player.

Increase Eye-to-Eye distance - Right, Down, Left, Up
Decrease Eye-to-Eye distance - Left, Down, Right, Up

 

Force2D mode:

A special mode - Force2D can be enabled if a 3D video is too stressful for eyes.

Toggle Force2D mode - Right, Left, Right, Left

 

Other functions:

Exit Program - Up, Left, Right, Down
