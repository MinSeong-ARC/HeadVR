HeadVR (https://github.com/autoandshare/HeadVR) is an open source VR video player for android based on VLC.


-- Nodding to play/pause, turning left/right to seek, ...
-- Support Local and network resources
-- Rich settings and a Force2D mode to make video more comfortable.
Note:
- Gyroscope sensor is necessary
- 360 video is not supported yet.


Usage:

There are 5 basic head motions: Up, Down, Left, Right, Hold still.

Basic functions:

Play/Pause - Down, Up (Nod)
Forward - Right, Hold still, Left (Cancel with any other action)
Rewind - Left, Hold still, Right (Cancel with any other action)
Next Video - Up, Down, Right, Left
Prev Video - Up, Down, Left, Right

 

Change settings:

Eye-to-Eye distance can be adjusted inside player.

Increase Eye-to-Eye distance - Right, Down, Left, Up
Decrease Eye-to-Eye distance - Left, Down, Right, Up

 

Force2D mode:

A special mode - Force2D can be enabled if a 3D video is too stressful for eyes. In this mode, same picture will be shown to both eyes.

Toggle Force2D mode - Right, Left, Right, Left

 

Other functions:

Exit Program - Up, Left, Right, Down
