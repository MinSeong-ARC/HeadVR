HeadVR (https://github.com/autoandshare/HeadVR) is an open source VR video player for android.



Features:

- Controlled by head motion, e.g., nod to play/pause. (Gyroscope sensor is necessary)

- Support playlist, viewing all the video's without taking off headset.

- Support most multimedia files with underlying libvlc.

- Local and network resources.

- Rich settings to make video comfortable to eyes.

- Force2D mode for stressful 3D video's.



Note:

- Currently there is no browsing function and please open a video with this app from a file manager/explorer.

- 360 video not supported yet.



Usage:

There are 5 basic head motions: Up, Down, Left, Right, Hold still.

 

Player basic functions:

Play/Pause - Down, Up (Nod)
Forward - Right, Hold still, Left
Backward - Left, Hold still, Right

 

Playlist functions:

If the video is a local file, the play list will be video files under the same directory.

A play list file (extension ".list") is also supported. For example, "movie.list":

movie.rmvb
dir1/movie1.mp4
dir2/movie2.avi

movie.list, movie.rmvb, dir1 and dir2 should be in the same directory.

Play list file can be on any network/cloud storage supported by file explorer.

Next Video - Up, Down, Right, Left
Prev Video - Up, Down, Left, Right

 

Change settings:

Some settings can be adjusted inside player.

Increase Eye-to-Eye distance - Right, Down, Left, Up
Decrease Eye-to-Eye distance - Left, Down, Right, Up

 

Force2D mode:

A special mode - Force2D can be enabled if a 3D video is too stressful for eyes. In this mode, same picture will be shown to both eyes.

Toggle Force2D mode - Right, Left, Right, Left

 

Other functions:

Exit Program - Up, Left, Right, Down
